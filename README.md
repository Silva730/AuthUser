# Sistema de Cadastro e Login de Usuários — Versão 1

Este projeto é um **sistema básico de cadastro e autenticação de usuários**, desenvolvido para estudo de backend e integração com banco de dados.  
A **versão 1** implementa segurança inicial, servindo como base para melhorias futuras.

## Objetivo

Demonstrar o fluxo completo de cadastro e login de usuários, incluindo validação de dados, criptografia de senhas e comunicação entre frontend e backend.

## Funcionalidades

- Cadastro de usuários
- Login com verificação de credenciais
- Armazenamento de usuários em banco de dados
- Criptografia de senhas com bcrypt
- Estrutura organizada e tipada

## Tecnologias utilizadas

### Backend
- Node.js
- Express
- TypeScript
- MariaDB
- bcrypt para segurança de senhas

### Frontend
- HTML

## Segurança (nível básico)

- Senhas armazenadas com hash (bcrypt)
- Queries organizadas para evitar erros comuns
- Sem proteções avançadas (rate limit, 2FA, etc.)
